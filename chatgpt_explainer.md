# 🚀 Project Overview: 3D Developer Portfolio

This project is a high-end, immersive developer portfolio built to stand out with interactive 3D elements and a space-themed aesthetic. It's designed to be more than just a static resume, offering a cinematic and engaging experience for visitors.

## 🌌 Core Aesthetic & Vibe
*   **Space Theme**: A dark cosmic canvas with floating particles, neon glows, and glassmorphism effects.
*   **Premium Animations**: Buttery smooth scroll, hover, and reveal effects powered by industry-leading animation libraries.
*   **Interactive Focus**: Elements react to the user, making the portfolio feel "alive" and responsive.

## ✨ Key Features
*   **Interactive 3D Keyboard**: A standout feature where a custom 3D keyboard scene (built in Spline) allows users to interact with skill keycaps. Each key reveals detailed skill information on hover or press.
*   **Smooth Navigation**: Uses Lenis for consistent, high-quality smooth scrolling across all browsers.
*   **Responsive Architecture**: Fully functional on mobile, tablet, and desktop, maintaining its premium look on all screen sizes.
*   **Dynamic Sections**: 
    *   **Hero**: Cinematic introduction with 3D depth.
    *   **Skills**: Categorized and visually indexed skills.
    *   **Experience & Projects**: Interactive cards for showcasing work and achievements.
    *   **Certifications**: Clean, organized display of professional growth.
    *   **Contact**: A modern form with validation and email delivery.
*   **Real-time Features (Optional)**: Support for live cursors, online visitor presence, and a global chat when connected to a compatible backend.

## 🛠️ Technology Stack (High Level)
*   **Frontend Framework**: Next.js 14 (App Router), React 18, and TypeScript for a robust, SEO-friendly architecture.
*   **Visuals & 3D**: Three.js and Spline for high-performance 3D rendering and interactive scenes.
*   **Interactions & Motion**: GSAP (GreenSock) and Framer Motion for complex, high-fidelity animations.
*   **Styling**: Tailwind CSS for rapid, modern design, alongside Radix UI for accessible UI primitives.
*   **Forms & Delivery**: Resend for seamless contact form email delivery.

## 🧩 Modularity & Rebranding
The project is built with customization in mind. All personal data (name, bio, social links, projects, and skills) is centralized in configuration files, allowing for easy rebranding without touching the underlying animation or 3D logic.
